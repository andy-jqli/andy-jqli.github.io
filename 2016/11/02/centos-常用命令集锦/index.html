Centos7常用命令

用户相关:

  • 使用whoami命令查看所使用的登录名称
  • 删除用户: userdel username
  • 切换用户: su – username
    [切换为username]
  • 用户配置文件存放路径:
    /etc/group 文件是用户组的配置文件。
    /etc/passwd 文件是用户的配置文件。

  • 修改系统语言为英文：
    vim /etc/sysconfig/i18n
    改为：LANG=”en_US.UTF-8”

  • 修改centos系统语言为英文（7）
    vi /etc/locale.conf
    改为：LANG=”en_US.UTF-8”

  • 查看centos是32位还是64位系统：
    sudo uname –m

网络相关：

  • 查看网卡uuid: #nmcli con show
  • 重启网络: service network restart
  • nmcli dev status
  • 添加默认网关路由: #route add default gw <默认网关地址>
  • 查看路由: route –n
  • 查看网关:netstat -rn
